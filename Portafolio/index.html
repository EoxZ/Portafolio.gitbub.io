Proyectos

Juego de laberinto con Python 🐍


Un juego de laberinto en Python, desarrollado completamente en consola, donde el jugador debe encontrar la salida aplicando lógica y control de movimientos.
🔍 ¿De qué se trata? El juego presenta un mapa en forma de matriz donde: # representa paredes ' ' (espacio en blanco) son caminos libres P es el jugador o persona E es la salida o meta que debe alcanzarse El jugador puede moverse por el laberinto usando comandos (W, A, S, D) y el sistema valida cada paso para evitar atravesar muros o salirse del mapa.

🛠️ Características destacadas:

  • ✅ Niveles predefinidos: fácil y medio
  • ✅ Generación aleatoria de laberintos (modo difícil)
  • ✅ Control total por teclado
  • ✅ Validación de colisiones y límites
  • ✅ Visualización clara del entorno
  • ✅ Contador de movimientos para evaluar rendimiento

👨‍💻 Tecnologías y conceptos aplicados:

  • 🔹 Python 3
  • 🔹 Funciones, bucles y estructuras de control
  • 🔹 Matrices (listas anidadas)
  • 🔹 Lógica espacial y control de eventos

Sistema de Gestión de Parqueadero en Java ☕


🎯Este proyecto simula el funcionamiento de un parqueadero real, permitiendo registrar la entrada y salida de vehículos, calcular tarifas en COP según el tipo de vehículo y controlar la capacidad disponible.

🔧 Características principales:

  • ✅ Registro de automóviles, motocicletas y camiones
  • ✅ Control de hora de entrada y salida
  • ✅ Cálculo automático del valor a pagar según tiempo y tipo de vehículo
  • ✅ Validación de datos y estructura modular
  • ✅ Aplicación de principios de POO (herencia, encapsulamiento, polimorfismo)

🧠 Tecnologías utilizadas:

  • 🔹 Java
  • 🔹 Apache NetBeans
  • 🔹 API java.time para control de tiempo
  • 🔹 Estructuras dinámicas con ArrayList
